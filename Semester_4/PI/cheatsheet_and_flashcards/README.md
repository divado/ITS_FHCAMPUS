# Cheat sheet and flashcards for the PI exam

**Cheat sheet** — one dense reference, grouped into seven themes (Foundations/Legal, Identity, eIDAS, Differential Privacy, Web Tracking, Anonymity/Location/AI/ML). Each item is trimmed to just the keywords and formulas you need to recall, with question numbers kept so you can cross-reference the full guide. Good for a final-hour skim or printing.

**Flashcards** — an interactive deck of all 35 cards. Click a card (or press Space) to flip between question and answer. Controls:
- ← / → or the buttons to move between cards
- **Shuffle** (or press S) to randomise, **Reset order** to go back to 1–35
- Tap the **☆** on an answer (or press F) to add a card to your **★ Review** deck, then switch to that mode to drill only the ones you flagged
- A progress bar and counter track where you are

Just open the HTML file in any browser. State (starred cards, shuffle) lives in the session, so it resets when you reload — that's intentional so you start fresh each study run.

One thing to flag: the flashcard answers are deliberately compressed for active recall, so they're terser than the study guide. If a card's short answer ever feels too thin for how your lecturer framed a topic, the full guide is the place to reconcile it — especially Q4 (which strategy taxonomy), Q17 (ZKP "envisioned" vs. the SD-JWT/mdoc the ARF actually mandates), and Q32/33 (the exact Art. 5 vs. Annex III split).
